# Reference Materials

## Prototypes

UI 原型檔案，用 HTML + inline CSS/JS 寫成，可直接在瀏覽器開啟。

| 檔案 | 說明 |
|---|---|
| `prototypes/v1.html` | 初始概念原型（Schema 列表 + 簡單欄位編輯）|
| `prototypes/v2.html` | 細化原型（加入命名建議 inline UI + Diff 視圖）|

## 使用方式

直接用瀏覽器開啟 HTML 檔案。無需伺服器，資料為靜態假資料。

這些原型是 UI 風格的參考，不是規格。實際功能以 `docs/SPEC.md` 為準。
